## 技术栈

* 前端：
  * React全家桶（React Hooks + React router + Redux/Mobx/Jotai）+ TypeScript
  * 前端工程化插件：Eslint/Prettier
  * Electron：桌面端应用打包
* 后端
  * nodejs
* 地图相关功能：百度地图API
* 部署：[Zeabur](https://zeabur.com)

## 运行方法

### 前端
